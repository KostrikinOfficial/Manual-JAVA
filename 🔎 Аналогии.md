## 🔎 Аналогии
___
### 📜 Язык программирования - язык, на котором пишут инструкцию.
### 💻 Программа - инструкция для компьютера
### ☕ Java - инструкция на языке Java
### 💬 javac - переводчик инструкции на языке Java на универсальный язык.
### 📄 Байткод - инструкция на универсальном языке
### ⚙️ JVM - переводчик универсального языка в язык конкретного процессора
### 🖥️ JRE - готовый автомобиль.
### 🛠️ JDK - автозавод.
### 📦 Переменная - подписанная ячейка камеры хранения.
### 📊 Тип данных - размер и тип коробки на складе.
